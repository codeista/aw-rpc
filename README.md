# aw-rpc

## Install venv

'>python3 -m venv flask-env'

## start env

'> . flask-env/bin/activate'

## Install requirements

'> pip install -r requirements.txt'

## Run locally

`> python3 app.py`

## Test

'./test.sh'


### View api doc/test

`localhost:5000/api/browse`
